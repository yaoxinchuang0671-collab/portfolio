# 潮汐视界 · 科技风作品集 Redesign 方案

> 设计方向：Cyber-Tech 赛博科技风  
> 核心体验：丝滑流畅、沉浸感、专业电竞视觉  
> 设计系统版本：v2.0

---

## 一、设计概念

### 1.1 风格定位：「暗物质引擎」
以深邃的宇宙暗物质为基底，辅以电光蓝紫能量流，营造"专业电竞视觉工作室"的科技感和力量感。不是花哨的霓虹灯堆砌，而是克制、精准、有张力的科技美学。

### 1.2 情绪板关键词
`深邃` · `精准` · `能量` · `流动` · `克制的光`

### 1.3 设计原则
| 原则 | 说明 |
|------|------|
| **暗为底** | 全局深色背景，内容通过光影层次浮现 |
| **光为引** | 用微光、边框发光、渐变引导视觉焦点 |
| **动为脉** | 每个交互都有反馈，滚动即叙事 |
| **简为核** | 信息密度适中，留白呼吸感充足 |

## 五、动效系统（核心：丝滑流畅）

### 5.1 缓动函数库
```css
--ease-out-expo:    cubic-bezier(0.16, 1, 0.3, 1);   /* 主要入场 */
--ease-out-quart:   cubic-bezier(0.25, 1, 0.5, 1);   /* 次要入场 */
--ease-in-out-sine: cubic-bezier(0.37, 0, 0.63, 1);  /* 循环动画 */
--ease-spring:      cubic-bezier(0.34, 1.56, 0.64, 1); /* 弹性反馈 */
```

### 5.2 动效时长规范
| 类型 | 时长 | 说明 |
|------|------|------|
| Micro | 150ms | 按钮反馈、颜色变化 |
| Short | 250ms | 悬停状态、小元素过渡 |
| Medium | 400ms | 卡片入场、布局切换 |
| Long | 600-800ms | 区块入场、页面过渡 |
| Ambient | 8-20s | 背景粒子、渐变流动 |

### 5.3 滚动触发动画（核心体验）

**ScrollReveal 配置：**
- 触发阈值：`threshold: 0.1, rootMargin: '0px 0px -50px 0px'`
- 入场方向：从下方 40px 淡入 + 轻微上浮
- 错开延迟：列表项每项 `delay += 80ms`
- 只触发一次，不重复

**视差层级（Parallax）：**
- 背景网格：scroll * 0.3（最慢）
- 装饰元素：scroll * 0.5
- 内容主体：scroll * 1（正常）
- 前景点缀：scroll * 1.2（最快）

### 5.4 微交互清单

| 元素 | 交互 | 效果 |
|------|------|------|
| 导航链接 | Hover | 下划线从中心向两边展开 + 文字颜色过渡到 Cyan |
| 作品卡片 | Hover | 边框出现渐变发光 + 图片微缩放 1.03 + 信息层上滑显示 |
| 分类标签 | Hover | 背景填充 + 文字变白 + 微上移 -2px |
| 按钮 | Hover | 背景色渐变偏移 + 微阴影扩散 |
| 滚动提示 | 持续 | 鼠标图标上下弹跳循环 |

**结构：**
```
┌────────────────────────────────────────────┐
│  [视频背景/粒子动画层]                        │
│  ┌──────────────────────────────────────┐  │
│  │  小标签：🎮 专业电竞视觉设计工作室       │  │

│  │                                      │  │
```

**设计细节：**
- 背景：动态粒子网格（Canvas）或视频，粒子随鼠标微移
- 标题：渐变文字（白→紫→青），letter-spacing 稍宽
- 副标题：Mono 字体，逐字打字机效果（已有，保留）
- 按钮：Ghost 风格，边框 hover 时渐变发光
- 滚动提示：底部居中，极简鼠标图标 + 脉冲动画

### 6.3 ② 分类导航设计

**顶部状态（初始）：**
- 水平排列的药丸形标签
- 背景：半透明毛玻璃 `backdrop-filter: blur(12px)`
- 活跃态：渐变填充 + 微阴影

**滚动后状态（吸顶左侧边栏）：**
- 固定左侧 200px 宽
- 垂直排列
- 背景：更深的半透明 + 右侧 1px 微光边框
- 当前分类左侧 3px 青色指示条
- 丝滑过渡动画：opacity 0→1 + translateX(-20px→0)，时长 400ms

### 6.4 ③ 作品瀑布流设计

**卡片设计（核心）：**
```
┌──────────────────┐
│                  │
│   [作品图片]      │  ← 圆角 16px，hover 微缩放
│                  │
│                  │
├──────────────────┤
│ 作品名称         │  ← 卡片底部信息层
│ #分类标签        │
└──────────────────┘
```

**卡片 Hover 效果：**
1. 图片：`scale(1.03)` + `filter: brightness(1.1)`，400ms
2. 边框：从透明渐变为 `linear-gradient(135deg, cyan, purple)`
3. 底部信息层：从下方滑入显示（之前隐藏或半透明）
4. 整体：`translateY(-4px)` + 微阴影扩散

**瀑布流参数：**
- 桌面端：3 列，gap 20px
- 平板：2 列
- 手机：1 列
- 无限滚动或分页加载

## 七、特效系统

### 7.1 背景粒子网格（Canvas）
- 粒子数量：60-80（视性能调整）
- 粒子颜色：低透明度青色/紫色 `rgba(0,240,255,0.3)`
- 连线规则：距离 < 150px 时绘制细线，透明度随距离衰减
- 鼠标交互：粒子向鼠标方向轻微偏移（磁吸效果，半径 200px）
- 动画：粒子缓慢漂移 + 鼠标交互 = 生动但不干扰

### 7.2 光晕扫过效果（Hero 标题）
- 标题文字上的光泽从左到右扫过
- 使用 `background-clip: text` + 渐变位置动画
- 每 5 秒循环一次，极 subtle

### 7.3 滚动进度条
- 顶部固定 2px 细线
- 颜色：从左到右的 cyan→purple 渐变
- 宽度随滚动百分比变化

## 九、技术实现建议

### 9.1 推荐技术栈
- **纯前端**：HTML5 + CSS3 + Vanilla JS（保持现有架构）
- **动画库**：GSAP + ScrollTrigger（丝滑动画核心）
- **粒子系统**：自研 Canvas（轻量可控）

### 9.2 性能要点
- 图片懒加载 + WebP 格式
- 粒子系统使用 `requestAnimationFrame`，页面不可见时暂停
- CSS 动画优先使用 `transform` 和 `opacity`
- 减少重排重绘

### 9.3 文件规划
```
index.html          ← 主页面（精简结构）
styles.css          ← 设计系统 + 全局样式
hero.css            ← Hero 区域专属样式
gallery.css         ← 作品展示 + 瀑布流
animations.css      ← 动效关键帧
particles.js        ← 粒子系统
scroll-effects.js   ← 滚动动画 + 视差
cursor.js           ← 自定义光标（可选）
```
